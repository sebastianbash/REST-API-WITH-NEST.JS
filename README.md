## Tech Stack


**Server:** Node, Express, Typescript, MongoDB 


## Run Locally

Clone the project

```bash
  git clone https://github.com/sebastianbash/REST-API-WITH-NEST.JS.git
```

Go to the project directory

```bash
  cd backen1
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```
